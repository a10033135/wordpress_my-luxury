== Description ==

NewsTwenty is a fast, clean, modern-looking Responsive News Magazine WordPress theme. The theme is fully widgetized, so users can manage the content by using easy to use widgets. NewsTwenty is suitable for dynamic news, newspapers, publishers, blogs, editors, online and gaming, newsportals,personal blogs, newspaper, publishing or review siteand any creative website. NewsTwenty is SEO friendly, WPML,Gutenberg, translation and RTL ready. Live preview : https://demo.themeansar.com/newsup and documentation at https://docs.themeansar.com/docs/newsup/
